## 初始化
操作：完全初始化，不完全初始化，如果没被初始化的指自动为0，不初始化所有值都是垃圾值，
清零 int a[5] = {0};

只有在定义数组类型的同时才可以整体赋值，其他情况下整体赋值都是错误的。

一个倒置函数：

    inta[7] = {1,2,3,4,5,6,7};
    
    int i,j;
    i = 0;
    j = 6;
    int t;
    while (i <j){
      t = a[i];
      a[i] = a[j];
      a[j] = t;
      i++;
      j--;
    }
